Usability Test Plan 3

Usability Test Plan Section

Test Plan for: Sinaesthetic

Persona Type Tested: Secondary

Context: School library
Duration: 50 minutes

User has been informed about the product and has signed a consent regarding this test. Background information can be viewed here.

Overall Objectives

Goals of this usability test:

  • Assess the overall ease of use of the product;
  • Assess the quality/usability of the speech recognition system;
  • Assess product accessibility in regards to visually impaired people;
  • Check tutorial effectiveness;
  • Assess if the product is helpful for a visually impaired person.
Scenario

Persona will draw using the speech system and recognize previously made drawings using the playback system.

Tasks

The tasks are (note: these need to be read out loud by interviewer):

Our product is open in a web browser. User will listen to a tutorial which provides instructions on how to use it.
  • Listen to the tutorial and complete it.
  • A drawing is opened inside the app. Please try to identify what is drawn using the sonar system.
  • Erase the drawing.
  • Please draw a simple flower using your voice.
Post-test Questions
  • Did you encounter any difficulties in completing the given tasks? If yes, please elaborate.
  • Did you find that the tutorial provided all necessary information to start using the product?
  • Do you believe the product would be useful in your day-to-day life?
Other Notes

Notes taken during the test:

  • - User was slightly confused and needed additional guidance even after hearing the tutorial;
  • - User completed the task in the estimated time;
  • - User could not easily identify the drawing given, but mentioned that it is possible after getting more familiar with the software;
  • - User could draw a flower using the voice system without issues;
  • - User believes the tool could help him in his day-to-day life and his studies (he studies chemistry).
  • - User expressed his frustration with not being able to see what he is doing, but mentioned that the app provides the tools necessary to deal with the situation and after some time he frustration would be reduced.
