Usability Test Plan Section

Test Plan for: Sinaesthetic

Persona Type Tested: Secondary

Context: School library
Duration: 40 minutes

User has been informed about the product and has signed a consent regarding this test. Background information can be viewed here.

Overall Objectives

Goals of this usability test:

  • Assess the overall ease of use of the product;
  • Identify obstacles in using the tools provided by the product;
  • Assess the quality/usability of the speech recognition system;
  • Assess if the product is enjoyable.
Scenario

Persona will draw some objects using both mouse and speech.

Tasks

The tasks are:

  • On the left side of the canvas, draw a red square using the tools from the app and your mouse.
  • On the right side of the canvas, draw a red square using your voice.
  • Erase the drawing.
  • On the left side of the canvas, draw a simple flower using the tools from the app and your mouse.
  • On the right side of the canvas, draw a simple flower using your voice.
  • Erase the drawing.
  • Close your eyes and try to draw the same flower using your voice.
Post-test Questions
  • Did you have fun while performing the tasks?
  • What is your opinion regarding this statement: "The menu items were well organized and functions were easy to find."?
  • Did you encounter any difficulties in completing the given task? If yes, please elaborate.
  • Do you prefer drawing using your mouse or your voice? Why?
  • Were the tools provided sufficient to perform the tasks? Would you have liked to have other tools available?
Other Notes

Notes taken during the test:

  • - User did not seem to have any trouble in using the app;
  • - User completed the task in the estimated time;
  • - User was smiling while drawing with her voice and stated "this is way more fun";
  • - User could reproduce the same drawing done with the mouse, using the voice recognition system;
  • - User managed to draw same flower with eyes closes with almost perfect accuracy;
  • - User believes the tool should offer more colours and some "picture effects".
