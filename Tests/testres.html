Usability Test Result Sheet

Test Results for: Sinaesthetic

Success Criteria

A successful design has been achieved when:

  • 80% of users find the application easy to use;
  • 50% of users find the speech system easy to use;
  • 70% of users find the tools provided sufficient for what they want to create;
  • 30% of users find the accessibility satisfying;
  • 30% of users find the application enjoyable;
  • 30% of users find the sonar system useful;
  • 50% of users find the application useful in their daily activities.

Summary

  • All users find the application easy to use;
  • All users find the tools and functions easy to find and understand;
  • 2 users find the speech system easy to use;
  • 1 user states that the sonar system is hard to understand at first and requires practice to get used to;
  • 1 user states that more colours and image effects should be added;
  • 2 users find the application useful in their daily activities;
  • 1 user states that the application is too simplistic and should be more complex;
  • All users are satisfied with the tutorials provided.

Post-test Questions Answers

Question: On a scale of 1 to 5, how did you find our software compared to other software you use normally for this kind of task, by ease of use? (1 means the most difficult to use, 5 means the easiest to use)

Expected answer: 3-4

Question: What is your opinion regarding this statement: "The menu items were well organized and functions were easy to find."?

Expected answer: Mostly true.

Question: Did you encounter any difficulties in completing the given task? If yes, please elaborate.

Expected answer: No.

Question: What are the must-have tools an app such as Sinaesthetic must provide in order to complete the given task?

Expected answer: Brush, colours, eraser.

Question: Would you use this app to complete some of your own tasks? Please give details regarding the task and how you might use our product.

Expected answer: Yes.

Question: Did you have fun while performing the tasks?

Expected answer: Yes.

Question: Do you prefer drawing using your mouse or your voice? Why?

Expected answer: Mouse, because it is faster or speech because it is more interesting.

Question: Were the tools provided sufficient to perform the tasks? Would you have liked to have other tools available?

Expected answer: Mostly yes.

Question: Did you find that the tutorial provided all necessary information to start using the product?

Expected answer: Mostly yes.

Question: Do you believe the product would be useful in your day-to-day life?

Expected answer: Yes.
