What is Sinaesthetic?

Sinaesthetic is a draw-by-sound web application. A sound recognition system takes user input, parses it and controls the application based on the recognised sounds. Take a look at the tutorial to get a glimpse of the Sinaesthetic prototype:

Questions, Options, Criteria

Q1: Where to position toolbar?
O: Left
O: Right
O: Top
O: Bottom
O: Left or Right and Top
O: Left or Right and Bottom
O: Left and Right
O: Top and Bottom
O: Invisible (tools activated by sounds only)
C: Familiar
C: Simmetric
C: Relevant tool grouping
C: Low user effort
C: Efficient screen space usage
C: Easy to use
C: Good canvas focus

Conclusion: Popular drawing software such as Microsoft Paint or Adobe Photoshop usually have the most used tools (pen, color selector etc.) in a toolbar on the left and some general tools (save, undo etc.) in a menu on top. This separation helps logically group tools and keeps the user in a familiar environment. However, the top menu is a main trait of desktop applications and the number of tools offered does not require cascading menu grouping. In order to save screen space and center the canvas for good focus, placing the drawing tools on the left to preserve familiarity and the general, utility tools on the right, seems like the best solution.

Q2: Type of sounds recognized?
O: Musical notes
O: Words
C: Intuitive
C: Language independent
C: High number of possible commands
C: Easy to use

Conclusion: While drawing by playing music is language independent and can create a high number of possible commands, using actual words to draw is more intuitive and easier to use while also not restricting the number of possible commands. Also adding the fact that it can be implemented easier and more reliable using voice recognition engines, drawing by using words seems to be the best solution.

Q3: How to draw using words?
O: Instruction based ("move left 3 units", "brush color red" etc.)
O: Based on the physical characteristic of the sound (pitch, length etc.)
C: Easy to use
C: Intuitive
C: High accuracy

Conclusion: Using exact instructions offers a much higher accuracy and ease of use and seems to be the best solution.
